# smallerEq
Returns true if the first argument is smaller than or equal to than the second.

#### Syntax
```
text <= text
boolean <= boolean
date <= date
number <= number
smallerEq(text, text)
smallerEq(boolean, boolean)
smallerEq(date, date)
smallerEq(number, number)
```

#### Examples
```
10 <= 8 == false
8 <= 8 == true
```
